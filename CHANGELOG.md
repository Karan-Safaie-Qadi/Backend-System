# Changelog

## v1.0.0 (2026-06-22)

### Initial Release

- Core Model extending PDO-Module with pagination, search, timestamps
- User management: register (email/phone mode), login, password reset
- Product management: CRUD, stock, pricing, categories, featured
- Article management: rich sections (text/list/table/image/mixed), TOC, publish workflow
- Role-based access control: user, admin, owner (extensible)
- Activity logging with user/entity context
- System service with custom methods, file upload, sanitization
- PHPMailer integration for email notifications
- Full database schema with 7 tables
- REST API layer for all operations
- Bilingual SPA test panel (English/Persian)
- PHPUnit test suite
